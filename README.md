## Featured work

### [World Cup 2026 · Forecast Engine](https://github.com/mohamedaziz-selmi/world-cup-2026-forecast)
A probabilistic model for the first 48-team World Cup: online **Elo** ratings → **Poisson** scorelines → **20,000 Monte-Carlo** tournaments over FIFA's official bracket, validated honestly with **walk-forward RPS** (not "did it guess the winner"). Ships with a designed, self-updating report — and runs entirely free, no API key.
